### SetUp

#### Local setup

1. install `node.js`

2. Open terminal and run `git clone https://github.com/pulkit-30/React-Blogging.git`

**Backend setup**

1. cd `/server`

2. run `yarn`

3. run `yarn start`

4. create `.env` file with following keys

```
DBURL=

```

5. restart server

**Frontend setup**

1. cd `/client`

2. run `yarn`

3. run `yarn start`
